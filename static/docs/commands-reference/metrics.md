# metrics

Manage metrics.

```sh
    usage: dvc metrics [-h] [-q] [-v] {show,add,remove} ...

    positional arguments:
        show                  Show metrics
        add                   Add metrics
        remove                Remove metrics

    optional arguments:
        -h, --help            show this help message and exit
        -q, --quiet           Be quiet.
        -v, --verbose         Be verbose.
```
